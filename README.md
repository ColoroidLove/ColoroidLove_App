### Coloroid Love

자신의 퍼스널컬러를 손쉽게 측정해보고 결과를 보여주는 앱 서비스 (2022.01 ~ )


### Features

* 카메라
    * 카메라를 통하여 자신의 퍼스널컬러를 체크 가능합니다.
* list view
    * 다른 유저들의 퍼스널 컬러도 확인가능합니다.
* Coloroid Love 웹페이지
    * Coloroid Love의 간단한 소개를 제공합니다.
